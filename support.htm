* How to patch                                                 * Patch (ver 1.41)

 

Hello, this is Abalon.

Patch version 1.41 is out.

The patch method is the same as the last patch.

Unzip the file into a temporary folder Patch141.exe When you run the file, it will be patched.

However, this patch executable file has an additional step to check the folder to be patched. We made this considering that some of the Dispel files are in multiple folders on users' computers. Please select the folder where Dispel is installed to ensure that the patch is installed in the correct folder.

After the patch is finished, you must delete the temporary folder and the unzipped files in it, as well as the original files! There will be no problems.

Even if you haven't installed the previous patch version, you only need to install this patch file. If you have any problems with the patch, please post on this bulletin board or DISCORD Please contact us and we will respond immediately.
